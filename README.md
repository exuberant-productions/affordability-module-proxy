# affordability-module-proxy

This is a proxy server for trulia home-detail page

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You will need to clone all three repository of modules under the project root folder.  
All dependencies listed in `package.json`.

### Installing

To install dependencies

```
npm install
```

## Development

In each module's repository(previously cloned), run node(express) server. There will be total of three servers running.  

To run server

```
npm run server
```
